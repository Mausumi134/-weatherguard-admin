# WeatherGuard Frontend

React admin panel for WeatherGuard Admin.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Environment

Copy `.env.example` to `.env` if the API URL is different from:

```text
http://localhost:3000
```

See the root README for the full architecture notes and demo flow.
